### CosmoExplorer

Aplicativo de estudo desenvolvido para explorar imagens e informações do universo usando APIs públicas como NASA APOD e SpaceX API. O app exibe a imagem astronômica do dia, permite visualizar explicações detalhadas e, futuramente, apresentará informações sobre lançamentos espaciais da SpaceX.
O projeto foi construído para praticar Jetpack Compose, consumo de APIs, navegação entre telas e boas práticas de arquitetura no Android.

### Funcionalidades
* Exibição da imagem do dia fornecida pela NASA APOD
* Informações como título, explicação e data
* Interface construída com Jetpack Compose
* Carregamento de imagens em alta resolução utilizando Coil
* Layout com rolagem vertical


### Tecnologias Utilizadas
* Kotlin
* Jetpack Compose
* Retrofit
* ViewModel
* Coroutines
* Flow
* Coil
* Bottom Navigation
* Material Design 3

### APIs Utilizadas
NASA APOD
* Astronomy Picture of the Day
* Retorna imagem, título, data e explicação científica

SpaceX API (em desenvolvimento)
* Retorna informações do lançamento mais recente
